## Installation
Add the JitPack repository in your build.gradle at the end of repositories:
```
    allprojects {
        repositories {
            ...
            maven { url "https://jitpack.io" }
        }
    }

```

```
    dependencies {
            compile 'com.github.Kunzisoft:Android-RememBirthDay:1.0'
    }
```
